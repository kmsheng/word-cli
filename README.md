# word-cli

## Installation

```bash
npm i -g word-cli
```

## Usage

```
  Usage: word [options] [command]


  Options:

    -V, --version  output the version number
    -h, --help     output usage information


  Commands:

    add [sentence] [description] [part]  Add a word data
    list                                 List words
    study                                Start to study words.
    rm                                   Remove word by id.
```
